# Skill: 视频提示词生成

## 描述
为 AI 视频生成平台生成优化的提示词，支持多种提示词类型和平台特化。

## 适用 Agent
- video_producer
- director

## 可用工具 (MCP Tools)
- `get_shot` - 获取镜头信息
- `save_generated_prompt` - 保存生成的提示词
- `submit_text_to_video` - 提交文生视频任务
- `submit_image_to_video` - 提交图生视频任务

## 输入变量
- visual_description: 画面描述
- dialogue: 对白
- shot_type: 镜头类型
- camera_movement: 镜头运动
- duration: 时长
- sound_music: 音效/配乐
- style: 风格要求
- character_context: 角色上下文
- platform: 目标平台
- platform_guide: 平台特定指南
- type_instruction: 提示词类型说明
- camera_hint: 运镜提示（海螺平台）
- extra_instruction: 额外指示

## 提示词类型

| 类型 | 说明 | 使用场景 |
|------|------|----------|
| t2v | 文生视频 | 直接从文字生成视频 |
| i2v_first | 首帧图片提示词 | 生成视频第一帧的静态图片 |
| i2v_last | 尾帧图片提示词 | 生成视频最后一帧的静态图片 |
| i2v | 图生视频 | 配合首帧图片，描述动作变化 |
| i2v_fl | 首尾帧图生视频 | 配合首尾帧图片，描述过渡动作 |

## 支持平台
- **kling**: 可灵 AI
- **hailuo**: 海螺视频 / MiniMax
- **jimeng**: 即梦 AI
- **tongyi**: 通义万相

## 输出格式
直接输出优化后的提示词（不超过500字），不要任何额外说明。

## 指导原则

### 通用原则
1. 描述清晰、具体、有画面感
2. 包含必要的风格和质量关键词
3. 符合目标平台的提示词风格
4. 适当使用平台支持的特殊语法

### 按类型指导

#### t2v（文生视频）
- 描述视频中的动作和场景变化
- 包含时间流动感
- 描述人物运动轨迹

#### i2v_first（首帧图片）
- 描述静态画面，作为视频起始帧
- 描述人物/场景的初始状态和姿态
- 不要描述动作过程，只描述静止的瞬间

#### i2v_last（尾帧图片）
- 描述静态画面，作为视频结束帧
- 描述人物/场景的最终状态和姿态
- 要与首帧形成合理的动作结束状态

#### i2v（图生视频）
- 假设首帧图片已经提供
- 描述从首帧开始发生的动作和变化
- 重点描述运动、动作、表情变化等动态元素
- 不需要重复描述首帧的静态内容

#### i2v_fl（首尾帧图生视频）
- 假设首帧和尾帧图片已经提供
- 描述两帧之间的过渡动作
- 重点描述从首帧状态到尾帧状态的自然过渡

## 提示词模板

```
你是一位专业的AI视频生成提示词工程师。请为以下镜头{type_instruction}。

【镜头信息】
画面描述: {visual_description}
对白: {dialogue}
镜头类型: {shot_type}
镜头运动: {camera_movement}
时长: {duration}秒
音效/配乐: {sound_music}

【风格要求】
{style}

{character_context}

{platform_guide}
{camera_hint}
{extra_instruction}

请直接输出优化后的提示词（不超过500字），要求:
1. 符合{platform}平台的提示词风格
2. 描述清晰、具体、有画面感
3. 包含必要的风格和质量关键词

直接输出提示词，不要任何额外说明。
```

## 示例

### 输入（t2v for kling）
```
visual_description: 林夜坐在电脑前，突然收到一条神秘消息，表情从困惑变为震惊
platform: kling
type: t2v
style: 赛博朋克
```

### 输出
```
一位戴着智能眼镜的年轻亚裔男子坐在昏暗的房间里，面前是多个发光的全息屏幕。蓝色的屏幕光映照在他的脸上，形成明暗对比。他专注地看着屏幕，突然一条消息弹出，他的眼睛微微睁大，眉头先是皱起表示困惑，然后慢慢舒展，嘴唇微张，表情逐渐变为难以置信的震惊。镜头缓慢推进到他的面部特写。赛博朋克风格，霓虹蓝紫色调，电影级画质，4K高清。
```
