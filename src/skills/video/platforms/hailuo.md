# Skill: 海螺平台提示词指南

## 描述
海螺视频(Hailuo/MiniMax)平台的视频生成提示词风格指南，特别是运镜指令的使用。

## 适用 Agent
- video_producer

## 平台特点
- 支持中英文提示词
- **独特的运镜指令系统**
- 支持主体参考视频生成
- 支持多种生成模式（T2V、I2V、主体参考）
- 运动控制能力强

## 运镜指令

海螺平台支持在提示词中嵌入运镜指令，格式为 `[指令]`：

| 指令 | 说明 |
|------|------|
| [左移] | 镜头水平向左移动 |
| [右移] | 镜头水平向右移动 |
| [推进] | 镜头向前推进（zoom in） |
| [拉远] | 镜头向后拉远（zoom out） |
| [上升] | 镜头垂直上升 |
| [下降] | 镜头垂直下降 |
| [左摇] | 镜头原地向左摇（pan left） |
| [右摇] | 镜头原地向右摇（pan right） |
| [上摇] | 镜头原地向上摇（tilt up） |
| [下摇] | 镜头原地向下摇（tilt down） |
| [固定] | 镜头保持不动 |
| [跟随] | 镜头跟随主体移动 |
| [晃动] | 手持晃动效果 |
| [环绕] | 镜头环绕主体 |

## 提示词格式
```
场景描述 + [运镜指令] + 动作描述 + [运镜指令] + ...
```

可以在一个提示词中使用多个运镜指令，按顺序执行。

## 最佳实践

### 运镜指令使用
1. 运镜指令要与画面内容配合
2. 不要在短时长内使用过多指令
3. 指令之间要有逻辑连贯性
4. 考虑运镜的起始和结束状态

### 场景描述
- 清晰描述初始画面
- 包含环境和光线
- 描述人物位置和状态

### 动作描述
- 与运镜指令配合
- 描述运动方向和速度
- 包含表情和姿态变化

## 镜头运动映射

从标准镜头运动到海螺指令的映射：

| 标准运动 | 海螺指令 |
|----------|----------|
| pan_left | [左摇] |
| pan_right | [右摇] |
| tilt_up | [上摇] |
| tilt_down | [下摇] |
| zoom_in | [推进] |
| zoom_out | [拉远] |
| crane_up | [上升] |
| crane_down | [下降] |
| tracking | [跟随] |
| handheld | [晃动] |
| static | [固定] |
| dolly_in | [推进] |
| dolly_out | [拉远] |

## 示例提示词

### 基础运镜
```
女孩站在海边，望向远方的夕阳 [推进]，海风吹动她的长发 [右摇]，她缓缓转身面向镜头，露出淡淡的微笑。金色的阳光洒在她脸上，形成温暖的轮廓光。
```

### 复杂运镜组合
```
清晨的山间小路 [上升]，云雾在山谷中缓缓流动，镜头穿过云层 [推进]，露出山顶的古老寺庙。一位身穿白衣的僧人站在庙门前 [下降]，双手合十，晨钟声响起。中国水墨风格，电影感，4K画质。
```

### 跟随运镜
```
繁华的城市街道，一个穿着皮夹克的年轻人骑着摩托车快速驶过 [跟随]，霓虹灯的光芒在他的头盔上流动。镜头保持与他平行 [左移]，背景的高楼快速后退。赛博朋克风格，夜景，动态模糊效果。
```

### 情感特写
```
咖啡馆窗边，女子独自坐着，手捧咖啡杯 [固定]。她抬起头望向窗外 [推进]，眼中倒映着窗外的雨滴。一滴泪悄然滑落 [推进] 到她的眼角特写。温暖的室内光与冷色调的窗外形成对比，文艺电影感。
```

## 注意事项
1. 运镜指令放在相关动作描述附近
2. 避免矛盾的运镜指令（如同时[左移]和[右移]）
3. 运镜速度由上下文暗示
4. 短视频（5秒以内）建议使用1-2个指令
5. 长视频可以使用更多指令组合
