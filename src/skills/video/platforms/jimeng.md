# Skill: 即梦平台提示词指南

## 描述
即梦(Jimeng)平台的视频生成提示词风格指南和最佳实践。

## 适用 Agent
- video_producer

## 平台特点
- 支持中英文混合提示词
- Pro版支持多镜头叙事
- 支持参考图片生成
- 丰富的风格预设
- 适合创意和艺术风格

## 提示词格式
```
详细的视觉描述 + 风格关键词
```

### 多镜头叙事格式（Pro版）
```
镜头1：场景描述；镜头2：场景描述；镜头3：场景描述
```

## 最佳实践

### 视觉描述
- 详细描述画面内容
- 使用具体的形容词
- 包含颜色、光线、质感
- 描述空间关系和构图

### 风格关键词
即梦平台支持丰富的风格关键词：

**艺术风格**
- 油画风格
- 水彩画
- 水墨画
- 素描
- 插画风格
- 动漫风格

**摄影风格**
- 电影感
- 纪录片风格
- 广告大片
- 人像摄影
- 风景摄影

**特殊效果**
- 延时摄影
- 慢动作
- 微距
- 航拍

**画质关键词**
- 高清
- 4K/8K
- 细节丰富
- 专业摄影
- 精细渲染

### 镜头语言
可以使用标准的镜头术语：
- 特写、近景、中景、远景、全景
- 俯拍、仰拍、平视
- 推拉摇移等运镜描述

## 示例提示词

### 单镜头风景
```
清晨的山间，云雾缭绕在翠绿的山峰之间，一只白鹤展翅飞过静谧的湖面，水面泛起层层涟漪。远处的古塔若隐若现，阳光透过云层洒下金色光芒。中国水墨画风格，意境悠远，4K高清。
```

### 多镜头叙事
```
镜头1：东京街头的雨夜，霓虹灯在湿润的地面上倒映出五彩斑斓的光芒；镜头2：一把透明雨伞下，一对年轻情侣相视而笑；镜头3：他们牵手奔跑在雨中，笑声回荡在空旷的街道。日系电影感，温暖色调，慢动作效果。
```

### 动漫风格
```
樱花盛开的校园，穿着水手服的少女站在樱花树下，粉色花瓣在她周围飘舞。她闭上眼睛，微风吹起她的长发和裙摆。日系动漫风格，细腻的光影效果，柔和的粉色调，梦幻般的氛围。
```

### 创意艺术
```
一只金色的鲤鱼在星空中游泳，它的鳞片反射出银河的光芒。鱼尾划过之处留下闪烁的星尘轨迹。超现实主义风格，梦幻，奇幻，高饱和度色彩，细节精致，8K画质。
```

### 商业广告风格
```
高端化妆品特写，金色的瓶身在柔和的灯光下熠熠生辉，细微的光斑在瓶身上流动。镜头缓缓旋转，展示产品的精致工艺。背景是纯净的白色，极简主义，广告大片质感，专业产品摄影。
```

## 注意事项
1. 多镜头叙事要保持场景连贯性
2. 风格关键词放在描述末尾
3. 中英文混合可以增强效果
4. 避免过于复杂的动作描述
5. 艺术风格作品可以更加抽象
