# Skill: 通义万相平台提示词指南

## 描述
通义万相(Tongyi Wanxiang)平台的视频生成提示词风格指南和最佳实践。

## 适用 Agent
- video_producer

## 平台特点
- 支持中文提示词（推荐）
- wan2.6模型支持多镜头叙事
- 描述要清晰具体
- 阿里巴巴旗下产品
- 适合商业和写实风格

## 提示词格式
```
清晰描述场景、人物、动作、氛围
```

## 最佳实践

### 场景描述
- 先描述场景环境
- 明确时间（白天/夜晚、季节等）
- 描述天气和光线条件
- 包含背景元素

### 人物描述
- 描述外貌特征
- 说明服装打扮
- 描述位置和姿态
- 包含表情状态

### 动作描述
- 使用清晰的动词
- 描述动作的方向和幅度
- 包含动作的起始和结束
- 注意动作的合理性

### 氛围描述
- 描述整体情绪基调
- 包含色调倾向
- 添加画面质量要求

## 推荐关键词

### 画质
- 电影感画面
- 高清
- 4K画质
- 专业摄影
- 细节丰富

### 光线
- 自然光
- 柔和光线
- 逆光
- 金色阳光
- 霓虹灯光

### 风格
- 写实风格
- 商业广告
- 纪录片感
- 电影感
- 温馨治愈

### 色调
- 暖色调
- 冷色调
- 高对比度
- 柔和色彩
- 复古色调

## 示例提示词

### 都市场景
```
城市街头，一个穿着白色连衣裙的女孩转身微笑，背景是霓虹灯闪烁的夜景。她的长发随着转身的动作轻轻飘起，脸上洋溢着幸福的笑容。都市夜景，电影感画面，高清画质。
```

### 自然风景
```
清晨的湖边，薄雾笼罩在平静的湖面上，远处的山峦若隐若现。一只白色的天鹅优雅地游过，在水面上留下淡淡的涟漪。阳光透过云层洒下，在湖面上形成金色的光斑。自然风光，宁静氛围，4K画质。
```

### 人物特写
```
咖啡馆内，一位戴眼镜的年轻男子坐在窗边，专注地看着笔记本电脑屏幕。阳光透过窗户洒在他的侧脸上，形成柔和的轮廓光。他时而皱眉思考，时而嘴角微微上扬。温暖的室内光线，文艺气质，近景镜头。
```

### 动态场景
```
海边沙滩上，一群年轻人在夕阳下奔跑嬉戏，他们的身影在金色的阳光中形成剪影。海浪轻轻拍打着沙滩，远处的太阳正在缓缓下沉，天空被染成橙红色。青春活力，温暖氛围，慢动作效果，电影感画面。
```

### 产品展示
```
精致的茶具放在木质茶盘上，热气从茶杯中缓缓升起。茶叶在透明的茶壶中舒展开来，颜色逐渐变深。柔和的侧光照亮茶具的细节，背景是模糊的中式元素。产品特写，高端质感，细节丰富，商业广告风格。
```

## 多镜头叙事（wan2.6）

wan2.6模型支持在一个提示词中描述多个连续镜头：

```
第一个镜头：城市的黎明，太阳从高楼间升起，金色的阳光洒在街道上。第二个镜头：早起的上班族走出地铁站，脚步匆匆。第三个镜头：咖啡店开始营业，店员打开门迎接第一位顾客。城市节奏，生活气息，纪录片风格。
```

## 注意事项
1. 使用清晰简洁的中文描述
2. 避免过于抽象或诗意的表达
3. 动作描述要符合物理规律
4. 多镜头叙事要保持时间逻辑
5. 人物特征描述要具体可视化
