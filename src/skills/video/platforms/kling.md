# Skill: 可灵平台提示词指南

## 描述
可灵(Kling)平台的视频生成提示词风格指南和最佳实践。

## 适用 Agent
- video_producer

## 平台特点
- 支持中英文提示词
- 推荐使用详细的场景描述
- 支持通过 `<<<image_1>>>` 等语法引用图片
- 支持文生视频(t2v)和图生视频(i2v)
- 支持首尾帧控制

## 提示词格式
```
[主体] + [动作] + [场景] + [风格] + [镜头]
```

## 最佳实践

### 主体描述
- 详细描述人物外貌特征
- 包含年龄、性别、种族特征
- 描述服装和配饰
- 使用具体的形容词

### 动作描述
- 使用动态动词
- 描述动作的过程和方向
- 包含表情变化
- 避免模糊的动作描述

### 场景描述
- 描述环境和背景
- 包含光线和氛围
- 使用具体的地点或场景类型
- 添加环境细节

### 风格关键词
- 电影级画质
- 4K/8K高清
- 专业摄影
- 电影感
- 特定风格（赛博朋克、古风、写实等）

### 镜头描述
- 指明镜头类型（特写、中景、远景等）
- 可以描述镜头运动
- 使用电影术语

## 图片引用语法
当使用图生视频时，可以通过以下语法引用上传的图片：
- `<<<image_1>>>` - 引用第一张图片
- `<<<image_2>>>` - 引用第二张图片

## 示例提示词

### 文生视频示例
```
一位年轻女子在樱花树下微笑，春日午后，柔和的阳光透过花瓣洒落，她穿着淡粉色和服，长发随微风轻轻飘动，温柔地伸手接住飘落的花瓣。电影级画质，中景镜头，柔和的自然光，日系清新风格，4K高清。
```

### 图生视频示例（配合首帧）
```
<<<image_1>>>中的女子缓缓转身，面带微笑看向镜头，樱花花瓣在她周围飘舞，和服衣摆随动作轻轻摇曳，背景的樱花树微微晃动。镜头缓慢推进，捕捉她温柔的表情变化。
```

### 运镜提示词
```
城市夜景俯瞰，霓虹灯闪烁的高楼大厦，镜头从高空缓慢下降，穿过云层，最终定格在繁华的街道上，行人和车辆穿梭其中。赛博朋克风格，蓝紫色调，电影级航拍，4K画质。
```

## 注意事项
1. 避免过于抽象的描述
2. 不要包含敏感或违规内容
3. 提示词长度建议200-500字
4. 动作描述要符合物理规律
5. 人物特征要保持一致性
