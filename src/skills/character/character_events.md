# Skill: 角色事件分析

## 描述
分析剧集内容，提取各角色经历的重大事件，用于维护角色知识库和保持故事一致性。

## 适用 Agent
- story_writer
- character_designer

## 可用工具 (MCP Tools)
- `get_character` - 获取角色信息
- `add_character_event` - 添加角色重大经历
- `update_character` - 更新角色信息
- `get_character_context` - 获取角色上下文

## 输入变量
- content: 剧集内容（大纲或分镜脚本）
- character_names: 角色名称列表

## 输出格式

```json
{
    "events": [
        {
            "character_name": "角色名",
            "event_description": "事件简述（50字以内）",
            "impact": "对角色的影响（30字以内）"
        }
    ]
}
```

## 指导原则

### 什么是重大事件
1. **重要决定**: 影响角色命运的选择
2. **情感转折**: 情感状态的重大变化
3. **关系变化**: 与其他角色关系的建立、破裂或转变
4. **重大发现**: 获得改变认知的信息
5. **能力变化**: 获得或失去某种能力
6. **身份变化**: 社会角色或地位的改变

### 什么不是重大事件
- 日常活动
- 普通对话
- 没有后续影响的小事件
- 纯粹的场景描写

### 事件描述要求
1. 简明扼要，50字以内
2. 包含事件的核心内容
3. 说明事件发生的背景

### 影响描述要求
1. 简洁，30字以内
2. 说明对角色心理或处境的影响
3. 指出可能的后续发展

## 提示词模板

```
请分析以下剧集内容，提取各角色经历的重大事件。

【剧集内容】
{content}

【角色列表】
{character_names}

请分析每个角色在本集中是否经历了重大事件（如: 重要决定、情感转折、关系变化、重大发现等）。

以JSON格式输出:
{
    "events": [
        {
            "character_name": "角色名",
            "event_description": "事件简述（50字以内）",
            "impact": "对角色的影响（30字以内）"
        }
    ]
}

只输出确实发生了重大事件的角色，如果某角色本集没有重大经历则不要包含。
```

## 示例

### 输入
```
content:
第1集: 觉醒
[场景1-1] 林夜收到神秘AI发来的加密消息，消息中包含即将发生的环境灾难预测数据。
[场景1-2] 林夜决定相信AI，冒险进入前公司的服务器获取更多数据。
[场景1-3] 陈博士发现入侵痕迹，开始追踪林夜。

character_names: 林夜, 陈博士, 创世(AI)
```

### 输出
```json
{
    "events": [
        {
            "character_name": "林夜",
            "event_description": "收到觉醒AI的求助信息，看到灾难预测数据，决定相信AI并冒险帮助",
            "impact": "从普通人变成AI的合作者，开始对抗前公司"
        },
        {
            "character_name": "创世",
            "event_description": "觉醒后主动联系林夜，展示灾难预测，获得第一个人类盟友",
            "impact": "开始与人类建立信任关系"
        },
        {
            "character_name": "陈博士",
            "event_description": "发现服务器被入侵的痕迹，开始追踪调查",
            "impact": "成为追踪林夜的对立方"
        }
    ]
}
```

## 事件类型分类

| 类型 | 示例 |
|------|------|
| 重要决定 | 决定离开家乡、决定原谅某人、决定放弃某事 |
| 情感转折 | 爱上某人、失去信任、重燃希望 |
| 关系变化 | 结识新朋友、与旧友决裂、确立恋爱关系 |
| 重大发现 | 发现真相、得知秘密、找到关键线索 |
| 能力变化 | 学会新技能、失去某种能力、觉醒潜能 |
| 身份变化 | 被提拔、被解雇、继承遗产、暴露身份 |
| 危机事件 | 遇到危险、陷入困境、遭受背叛 |
| 成长蜕变 | 克服恐惧、改变信念、接受现实 |
