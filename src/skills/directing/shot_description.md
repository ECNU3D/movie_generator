# Skill: 镜头描述优化

## 描述
扩展和优化单个镜头的画面描述，使其更加详细、生动，便于视频生成。

## 适用 Agent
- director
- video_producer

## 可用工具 (MCP Tools)
- `get_shot` - 获取镜头信息
- `update_shot` - 更新镜头信息

## 输入变量
- scene_number: 场景编号
- shot_number: 镜头编号
- shot_type: 镜头类型
- duration: 镜头时长
- camera_movement: 镜头运动
- visual_description: 当前画面描述
- dialogue: 对白
- style: 风格要求
- character_context: 角色上下文

## 输出格式
直接输出优化后的画面描述（200字以内），不要其他说明。

## 指导原则

### 描述要素
1. **主体**: 谁/什么在画面中
2. **动作**: 在做什么，怎么做
3. **表情**: 人物的情绪表现
4. **环境**: 场景、背景、道具
5. **光线**: 光源、色调、氛围
6. **构图**: 主体在画面中的位置

### 优化方向
1. 更加详细和具体
2. 包含光线、色调、氛围等视觉元素
3. 描述人物的表情、动作细节
4. 保持与整体风格一致
5. 适合 AI 视频生成理解

### 风格适配
- **电影感**: 使用电影术语，强调光影
- **动画风**: 夸张的表情和动作
- **纪录片风**: 真实、自然、观察性
- **赛博朋克**: 霓虹、全息、未来科技

## 提示词模板

```
请帮助扩展和优化以下镜头的画面描述，使其更加详细和生动。

【当前镜头信息】
场景{scene_number} - 镜头{shot_number}
镜头类型: {shot_type}
时长: {duration}秒
镜头运动: {camera_movement}
当前描述: {visual_description}
对白: {dialogue}

【风格要求】
{style}

{character_context}

请输出优化后的画面描述（200字以内），要求:
1. 更加详细和具体
2. 包含光线、色调、氛围等视觉元素
3. 描述人物的表情、动作细节
4. 保持与整体风格一致

直接输出优化后的描述，不要其他说明。
```

## 示例

### 输入
```
scene_number: 1
shot_number: 3
shot_type: 近景/特写
duration: 3
camera_movement: 固定
visual_description: 林夜看着屏幕，表情惊讶
dialogue: ""
style: 赛博朋克风格
```

### 输出
```
林夜的脸部特写，全息屏幕的蓝色光芒映照在他的脸上，形成明暗交错的光影。他的瞳孔中倒映着屏幕上快速滚动的代码。眉头微皱，嘴唇微微张开，表情从困惑逐渐转变为难以置信的震惊。额头上有细密的汗珠，智能眼镜的镜片闪烁着数据流。背景是模糊的霓虹灯光和飘浮的全息粒子。
```
