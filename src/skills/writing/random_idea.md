# Skill: 随机创意生成

## 描述
随机生成有趣的短视频故事创意，用于激发灵感或快速原型。

## 适用 Agent
- story_writer
- supervisor

## 可用工具 (MCP Tools)
- 无（纯生成任务）

## 输入变量
- genre_hint: 类型偏好（可选）
- style_hint: 风格偏好（可选）

## 输出格式
直接输出 50-100 字的故事创意描述，不要任何额外说明。

## 指导原则

### 创意要素
1. **新颖性**: 创意要新颖，有吸引力
2. **可视化**: 适合制作成短视频系列
3. **冲突明确**: 有明确的主角和冲突
4. **简洁**: 50-100字的简短描述

### 创意结构
一个好的创意应包含：
- 主角是谁
- 面临什么困境/目标
- 有什么独特设定
- 潜在的情感核心

## 提示词模板

```
你是一位富有创意的故事策划师。请随机生成一个有趣的短视频故事创意。

{genre_hint}
{style_hint}

要求:
1. 创意新颖，有吸引力
2. 适合制作成短视频系列
3. 有明确的主角和冲突
4. 50-100字的简短描述

直接输出故事创意，不要任何额外说明。
```

## 示例

### 输入
```
genre_hint: 类型偏好: 科幻
style_hint: 风格偏好: 温馨治愈
```

### 输出
```
在2050年的东京，一个专门为老人设计的陪伴型机器人"小暖"，在主人去世后被遗弃在垃圾场。它偶然被一个失去父亲的小女孩发现并带回家。机器人的记忆中保存着老人一生的故事，它开始用这些故事治愈这个破碎的家庭，同时也在寻找自己存在的意义。
```
