AI电影生成器

我想做一个AI电影生成器，下面是我记录的一些想法和需求。首先生成一部电影最主要的是视频分镜的生成，而视频分镜的生成又依赖于图片分镜的生成。同时由于现在主流平台主要支持几种视频生成模式：
1. 图生视频：用户提供首帧，或首尾帧，加上文字提示词
2. 文生视频：用户仅提供提示词来控制视频内容的生成
3. 主体参考：上传主体角色的图片，保持人物的一致性。

我用了很多主流平台：
可灵：https://app.klingai.com/cn/
通义万相：https://app.klingai.com/cn/
即梦AI：https://jimeng.jianying.com/ai-tool/home/
海螺视频：https://hailuoai.com
Elser AI：https://www.elser.ai/zh/home

我觉得Elser AI是最接近我想要的，因为他有完整的剧本生成-》角色生成-》分镜生成-》故事版生成，但是它的收费很贵，所以我想自己做一个。我希望做一个开源的，可以接入可灵，通义万相，即梦AI和海螺视频，因为他们都提供了API接口。剧本的生成我觉得可以接入阿里的千问平台，豆包，智普平台，ChatGPT和Gemini。角色生成，也就是角色人物设定图（多个角度），可以接入文生成图模型，我前面说的这几个平台也应该都支持。

我希望做的一个特色功能就是可以每一个功能让用户使用不同模型供应商的模型和API。同时，用不同的方法来实现视频的前后一致性。由于主体参考的成本很高（也不是所有平台都支持主体参考，主体参考支持的数量也有所不同），所以我们可以支持主体参考，但是也可以支持低成本的，通过提示词来控制的一致性。

另外就是我们需要非常好的AI剧本生成编辑器，让用户可以生成，编辑管理很长的剧本（最终成片长达一到三小时）

别的功能比较常规，就是可以让用户有效的管理素材，管理剧本，管理生成的视频。

技术上我希望前端使用nextjs，后段使用python fastapi，数据库先用sqllite就行了。
