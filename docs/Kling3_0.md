FAQ
支持输入的素材
● 图片：支持你上传最多 7 个图片宽高尺寸不小于 300 px，文件大小  ≤ 10MB，格式为 .jpg / .jpeg / .png 的图片
● 视频：支持你上传一个 3s ≤ 时长 ≤ 10s，文件大小 ≤200MB，720p ≤ 分辨率 ≤ 2k 的视频
● 主体：
	○ （1）支持你上传/使用 AI 生成多个不同视角的图片（最多 4 张），组合为一个主体，为模型提供更丰富的参考信息。但主体为角色类型时，你还可以上传一段 5-30s 的单人说话音频（建议背景声干净、语速适中、情绪和风格统一的中性语音），为这个角色绑定一个音色。
	○ （2）支持你上传一个 3-8s 的单人角色类视频片段，创建一个更生动、信息更丰富的视频角色主体。视频中的人声可绑定为该角色的音色。
备注：当输入区存在视频时，图片/主体合计最多可上传 4 个；不存在视频时，图片/主体合计最多可上传 7 个。
